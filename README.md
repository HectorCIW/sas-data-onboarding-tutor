# SAS Data Onboarding Tutor

Public static teaching site for beginner SAS data onboarding practice.

## Site

- `index.html`: public entry page
- `teaching-site/index.html`: lesson workspace
- `teaching-site/quiz.html`: SAS Studio-like quiz lab

## Learning Sources

The lesson workspace references SAS Help Center, SAS GitHub examples, UCLA OARC
modules, PHUSE examples, and The Little SAS Book resources. Links are shown
inside each lesson as learning references; no copyrighted book text is copied
into this site.

## Data Boundary

This repository is for synthetic teaching examples only. Do not publish PHI,
PII, credentials, real patient records, or protected clinical data here.

Browser checks are static teaching checks. Runtime validation still requires
SAS 9.4, SAS OnDemand, or SAS Viya plus a reviewed SAS log.

## Deployment

GitHub Pages publishes from the `main` branch root.
